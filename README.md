Code for "Star-formation variability on the star-forming main sequence during the
Epoch of Reionization"
----------

**Paper Authors:** H. T. J. Bevins, S. Tacchella, C. Simmonds

Code to model and fit power spectral densities for star formation to measurements of the intrinsic scatter around the main sequence from a survey of approximately 17000 galaxies in [Simmonds et al. 2025](https://arxiv.org/abs/2508.04410).
